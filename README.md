# Local Media Shelf

Local Media Shelf is a small browser-based media library built as a web development capstone project. The finished app will let a user browse local media, view title details, track watched and favorite status, and look up movie and TV metadata from TMDB.

## Planned Features

- Browse and filter a small media catalog stored in JSON.
- Open a details/watch page for a selected title.
- Track watched and favorite status.
- Look up movie and TV metadata through the TMDB API.
- Use a responsive layout on mobile and desktop screens.

## Requirements

- Node.js 20 or newer
- npm

## Setup

1. Clone the repository.
2. Run `npm install`.
3. Copy `.env.example` to `.env` and add a TMDB API key.
4. Run `npm start`.
5. Open `http://localhost:3000` in a browser.

The repository does not include copyrighted media. A legal browser-compatible sample video can be added later.
